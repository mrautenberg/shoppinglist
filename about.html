About the project

This is a simple shopping list created for educational purposes.
My plan is to make it possible to add items both by typing, speech and scanning a hand written list.
The first step is to make it possible to store the shopping list in local storage, and the second step will be to add functionality to save and share shopping lists with your google account.
The shopping list is a work in progress, with many features yet to be done :)
Check out my GITHUB to see more of my projects.
Enjoy!
