About the project

This is a simple shopping list done for educational purposes.
My plan is to make it possible to add items both by typing, speech and scanning a hand written list.
The shopping list is only stored in local storage for now, but I plan to make it possible to save and share shopping lists with a google account.
The shopping list is a work in progress, with many features yet to be done :)
Check out my GITHUB to see more of my projects.
Enjoy!
